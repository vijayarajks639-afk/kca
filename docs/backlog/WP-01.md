# WP-01 — Repo scaffold + CLAUDE.md + CI

**Epic:** E1 Foundation · **Sprint:** Sprint 0 · **Package:** `/ and .github/` · **Depends on:** none
**Branch:** `wp-01-repo-scaffold-claude`

## Scope

Monorepo layout; CLAUDE.md rules; ruff/pytest CI; PR template with acceptance-criteria checklist. (Mostly done by the starter zip — verify and finish.)

## Acceptance criteria (PR checklist — all must pass)

- [ ] Fresh clone: `make up` then `make test` passes
- [ ] CI green on empty packages
- [ ] CLAUDE.md reviewed by architect (the user)

## Rules reminder

Follow CLAUDE.md strictly: LLM in L3/L4 only · no LLM-computed numbers · retrieval takes as_of + caller identity, permission filter pre-ranking, fail-closed · every model call in the ledger · cross-package via contracts/ only · no cloud SDKs in core · tests ship with the WP.

## Out of scope

Anything not listed above. Do not refactor other packages, add dependencies beyond pyproject, or change contracts/ schemas without flagging it in the PR description.
